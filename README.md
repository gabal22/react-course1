# Project Repository

Welcome to the project! Please note that the `main` branch is currently used for repository management and configuration. All active code and development live within dedicated feature and environment branches.

## Branch Overview

* **`develop`**: The primary integration branch where ongoing features are merged and tested.
* **`popcorn`**: A specialized feature or release branch created directly from `develop`.

## How to Access the Code

To view or work with the code, switch to the appropriate branch using the GitHub UI dropdown or your local terminal:

```bash
# Fetch all branches from remote
git fetch origin

# Switch to the develop branch
git checkout develop

# Or switch to the popcorn branch
git checkout popcorn
```
